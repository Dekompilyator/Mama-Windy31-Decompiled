Open game in unity versions between 2020 - 2021 (in 2018 - 2019 game brokes). After openning project, go to package manager and install package Advertisement. After this find script called SmartInputManager, open it and change to 

public enum EInputType	
{
Standalone = 1,
TouchControlsKit = 0
}

NOTE: Buttons and some other images in menu can have some wrong scale and size values, this is not critical, you can fix this yourself n 10 seconds.
